# Frontend Pages Directory
This directory contains individual HTML views for different pages of the B2B Bulk Order Portal.

## Expected Pages:
- `login.html`: Auth page for B2B buyers and admins.
- `dashboard.html`: Main dashboard to view orders, statuses, and quick metrics.
- `order-sheet.html`: Direct excel/CSV sheet order template parser.
- `catalog.html`: Browsable product catalog with bulk discount tier visualizers.
